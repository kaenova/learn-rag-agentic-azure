# Learn Agentic RAG Azure

A place where i learn RAG using Azure Stack. This includes:
- Azure AI Search
- Azure OpenAI
- (stil learning...)

This project use `uv`. Install `uv` first and then
```sh
$ uv venv --python 3.12

$ source .venv/bin/activate

$ uv sync

# Run the jupyter lab
$ uv run --with jupyter jupyter lab
```
